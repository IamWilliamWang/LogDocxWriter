# Word 记事本 v3.9
基于.NET的记事本风格的Word文档编辑器

## 基本介绍
记事本风格的Word文本编辑器，支持docx、txt编辑功能。支持修改字体字号，窗口置顶，自动保存、备份，暗黑模式，查找与替换，插入链接等高级功能。
